mermaid是一种用于绘制图表的语言，可用于markdown
graph LR或graph TD确定图的指向。具体语法可以根据下述代码即生成图像判断
#### 流程图



````
```mermaid
graph LR
A[方形] -->B(圆角)
    B --> C{条件a}
    C -->|a=1| D[结果1]
    C -->|a=2| E[结果2]
    F[横向流程图]
```
````
